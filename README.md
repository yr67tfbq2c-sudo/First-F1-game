# formula 1 turn-based racing game 
A turn-based racing game build in python that brings out the fun of Formula 1 through an interactive command lines interface.

# About this project
This game stimulates F1 racing mechanics and complexity in a turn-based format allowing players to make strategic decision to decide the outcome of the race. The player makes strategy calls and compete against opponents in an engaging racing experience.

# Features
-  Turn-based gameplay mechanics
- Strategic decision-making system
- Race simulation with multiple competitors
- Command-line interface for easy accessibility
- Object-oriented design for scalability

# Technology used
- python3.x - the core programming language
- Input validation and error handling

# How to play/run
1. Make sure you have python istalled - the latest version
2. Clone this repository:
```bash
   git clone https://github.com/yr67tfbq2c-sudo/First-F1-game.git
```
3. Navigate to the project directory:
```bash
   cd First-F1-game
```
4. Run the game:
```bash
   python f1_game.py
```
